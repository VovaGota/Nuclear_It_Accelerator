# Аннотация
  В ходе данной выполнения данного задания от команды требовалось согласовать параметры пучка с каналом ускорителя. Показать, из каких элементов магнитной системы состоит согласующий участок. Представить визуализацию изменения эмиттанса пучка в согласующем участке.

# Основная часть
  Командой было предложено решение в виде создания линейного ускорителя с основным применением квадрупольных магнитных линз и дрейфовых элементов для участков между квадрупольными линзами (продолжение в файле отчета)
